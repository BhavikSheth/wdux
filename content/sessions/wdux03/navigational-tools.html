Introduction

  • Allow user to get around a Web site
  • Provide orientation to users
  • Easily distinguished from content
  • Function should be intuitive and obvious
  • Use standard navigational tools
    • What are standard navigational tools?

Standard Navigational Tools

  • Navigation bars or menus (global and local)
  • Mega drop-downs (e.g. Argos)
  • Home button / link
  • Footer
  • Search
  • Breadcrumb trail (e.g. AES)
  • Tabs (e.g. Reuters)
  • Site map (e.g. TV Licensing)
  • Site index (e.g. LSE)
  • Links!

Standard Navigational Tools

  • Not all navigational tools will be appropriate for every site
  • Consider each when designing a site
  • Provide alternatives / duplicates
  • Some common conventions:
    • Logo in top left links to the home page
    • Avoid linking to the page you're already on
    • Contact and T&C links in the footer
