Brackets

Brackets is an open-source code editor for designers and frontend developers

Available for Windows, Mac & Linux

  • HTML
  • CSS
  • JavaScript

Download: brackets.io

Features

  • Code completion
  • Syntax highlighting
  • Error checking
  • Live editing
  • Plugins/extensions

Emmet

Toolkit for Web developers

  • Shortcuts/expressions to generate chunks of HTML
  • Extension for Brackets
    1. Select: File > Extension manager
    2. Search for: emmet
    3. Install
  • General usage: type in expression in Brackets (e.g. h1) and trigger expressions by pressing the Tab key
  • Must be editing an HTML document; if it's a new document, save with a .html extension first

Emmet Examples

Save the file with a .html extension first

  • doc = Basic HTML page structure
  • html:4t = HTML 4 page structure
  • html:5 = HTML5 page structure
  • html:xs = XHTML page structure
  • ul>li*5 = Create an unordered list with five items
  • table>tr>td = Table with one row and one column
  • table>tr*3>td*2 = Table with three rows and two columns
  • img = Image element
  • a = Hyperlink
