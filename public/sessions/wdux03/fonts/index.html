Font Size Best Practice

Best practice in sizing text using CSS:

  • body { font-size: 100% }
    (otherwise older versions of IE exaggerate sizes)
  • Apply font-size to other elements in ems
  • Allows all users to resize text
  • Works across all browsers

Downside Of Ems

  • Size of em is relative to parent
  • li { font-size: 0.5em } will be applied successively to nested lists
  • One day will switch to rems
  • rem allows sizing relative to root element
  • Only works in current browsers, so need backups (for IE)
  • More: EM vs REM vs PX & MDN font-size

Web Fonts

  • Allow using fonts not installed on user's computer
  • Google Fonts is easy to use, but limited
  • Alternatively, use own font (e.g. downloaded from Font Squirrel) and apply using @font-face:
    • @font-face {
      font-family: "Quicksand";
      src: url("quicksand.otf");
      }
    • body {
      font-family: "Quicksand", sans-serif;
      }
